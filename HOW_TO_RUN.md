## üöÄ Method 1: Quick Start (Recommended for First-Time Users)

### Windows Users

1. **Open PowerShell in the app folder**
   ```powershell
   cd "C:\Users\ksesh\OneDrive\Desktop\New Folder\New Folder\app"
   ```

2. **Run the setup script**
   ```powershell
   .\setup.bat
   ```
   
3. **The script will**:
   - Check Python installation
   - Create virtual environment
   - Install dependencies
   - Set up .env file
   - Open .env for you to add your API key

4. **Add your OpenAI API key to .env**
   ```
   OPENAI_API_KEY=sk-your-actual-key-here
   ```
   Save and close the file.

5. **Run the application**
   ```powershell
   venv\Scripts\activate
   streamlit run streamlit_app.py
   ```

### macOS/Linux Users

1. **Open Terminal in the app folder**
   ```bash
   cd /path/to/app
   ```

2. **Make setup script executable and run**
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```

3. **Add your OpenAI API key to .env**
   ```bash
   nano .env
   # Add: OPENAI_API_KEY=sk-your-actual-key-here
   ```

4. **Run the application**
   ```bash
   source venv/bin/activate
   streamlit run streamlit_app.py
   ```

## üîß Method 2: Manual Setup (Step-by-Step)

### Step 1: Check Python Version

```bash
python --version
# Should show Python 3.8 or higher
```

If Python is not installed:
- Windows: Download from [python.org](https://www.python.org/downloads/)
- macOS: `brew install python3`
- Linux: `sudo apt install python3 python3-pip`

### Step 2: Navigate to Project

```bash
cd "C:\Users\ksesh\OneDrive\Desktop\New Folder\New Folder\app"
```

### Step 3: Create Virtual Environment

**Windows:**
```powershell
python -m venv venv
venv\Scripts\activate
```

**macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

You should see `(venv)` in your terminal prompt.

### Step 4: Install Dependencies

```bash
pip install -r requirements_streamlit.txt
```

This installs:
- streamlit==1.31.0
- openai==1.12.0
- python-dotenv==1.0.1
- pandas==2.2.0
- python-dateutil==2.8.2

### Step 5: Configure Environment

```bash
# Copy example file
copy .env.example .env  # Windows
cp .env.example .env    # macOS/Linux

# Edit .env file
notepad .env  # Windows
nano .env     # macOS/Linux
```

Add your OpenAI API key:
```
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxx
```

**Get your API key**: https://platform.openai.com/api-keys

### Step 6: Run Application

```bash
streamlit run streamlit_app.py
```

The browser will open automatically at `http://localhost:8501`

## ‚òÅÔ∏è Method 6: Deploy to Cloud (Streamlit Cloud)

### Prerequisites
- GitHub account
- Code pushed to GitHub repository

### Steps

1. **Push code to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/talentscout-ai.git
   git push -u origin main
   ```

2. **Visit Streamlit Cloud**
   - Go to [share.streamlit.io](https://share.streamlit.io)
   - Sign in with GitHub

3. **Create New App**
   - Click "New app"
   - Select your repository
   - Branch: main
   - Main file path: `streamlit_app.py`

4. **Add Secrets**
   - Click "Advanced settings"
   - Add:
     ```toml
     OPENAI_API_KEY = "sk-your-key-here"
     ```

5. **Deploy**
   - Click "Deploy!"
   - Wait 2-3 minutes
   - Get your public URL: `https://yourapp.streamlit.app`

## üîç Troubleshooting

### Issue: "ModuleNotFoundError: No module named 'streamlit'"

**Solution:**
```bash
# Make sure virtual environment is activated
venv\Scripts\activate  # Windows
source venv/bin/activate  # macOS/Linux

# Reinstall dependencies
pip install -r requirements_streamlit.txt
```

### Issue: "OpenAI API Error"

**Possible causes:**
1. API key not set in .env
2. Invalid API key
3. API key quota exceeded

**Solution:**
1. Check `.env` file exists and has `OPENAI_API_KEY=sk-...`
2. Verify key at https://platform.openai.com/api-keys
3. Check your OpenAI account billing

**Temporary workaround:**
The app has fallback questions that work without API key!

### Issue: "Port already in use"

**Solution:**
```bash
# Use a different port
streamlit run streamlit_app.py --server.port=8502
```

### Issue: "Python not found"

**Solution:**
- Windows: Install from python.org
- Ensure "Add Python to PATH" was checked during installation
- Try `python3` instead of `python`

### Issue: "Permission denied" on setup.sh

**Solution:**
```bash
chmod +x setup.sh
./setup.sh
```

### Issue: Virtual environment not activating

**Windows PowerShell execution policy error:**
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
venv\Scripts\activate
```
